# Vimrc
My *neovim* configs.

`init.vim` - Config in Vimscript

To install this config:

```bash
git clone git@github.com:stepan-zubkov/vimrc.git
cd vimrc/
chmod +x install.sh
./install.sh
```

To update config in repository:

```bash
chmod +x update.sh
./update.sh
```
